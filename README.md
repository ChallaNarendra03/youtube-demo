# MiniTube 🎬

MiniTube is a YouTube-like video platform where users can:

- Register and login
- Upload videos
- Watch videos
- Delete videos
- Search videos
- Navigate with sidebar

## Live Demo

https://youtube-demo-4.onrender.com

## Tech Stack

Frontend
- HTML
- CSS
- Bootstrap
- JavaScript

Backend
- Python
- Flask

Deployment
- GitHub
- Render Cloud

## Features

- User Authentication
- Video Upload
- Video Playback
- Video Delete
- Search Bar
- Sidebar Navigation
- Responsive UI
